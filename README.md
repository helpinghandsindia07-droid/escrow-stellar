# escrow-stellar
Escrow Smart Contract (Stellar Testnet)
This repository contains an escrow smart contract deployed on the Stellar testnet. It is intended for holding funds between a payer and a payee until predefined conditions are met.

Testnet contract on StellarExpert:
CCBVLGIVYSL2ABSWSP2OKUNVUONYZ4FT4KNHCT3DN6PIDJR2TYMPOTE6

Overview
The contract is deployed as a Soroban WASM smart contract on the Stellar testnet and can be called via Soroban RPC or any supported SDK.

You can integrate it into dApps where a neutral on-chain escrow is required between two or more parties.

Typical escrow flows you can support with this contract:

Payer locks funds into the escrow contract.

Parties agree on terms off-chain or via extra on-chain data.

Upon fulfillment (or dispute resolution), funds are released to the payee or returned to the payer.
